# -Basic-Color-Palette
A basic color palette UI design created in Figma, exploring fundamental color combinations, swatches, and visual harmony for UI/UX projects.
 demo video https://youtu.be/TgxO8IsNeoY This project presents a Basic Color Palette design built in Figma. It covers foundational color theory concepts and visual design, including:

Primary, secondary, and accent color swatches
Color codes (HEX / RGB values)
Light and dark tone variations
Harmonious color combinations for UI use
Clean, minimal Figma layout for easy reference
Useful as a starting reference for any UI/UX design project or frontend development work
